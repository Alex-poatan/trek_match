Descripción del libro

Aunque los inicios de Internet se remontan a los años sesenta, no ha sido hasta los años noventa cuando, gracias a la Web, se ha extendido un uso po todo el mundo. En pocos años, la Web ha evolucionado permanentemente con la aparición de páginas sencillas, con pocos imágenes y contenidos diversos que provienen de bases de datos o a partir de contenidos únicos con usuarios o a partir de contenidos únicos con usuarios o a partir de contenidos únicos con usuarios.

Todos los sitios Web están constituidos con la misma tecnología, con el Lenguaje de marcas de hipertexto (Hypertext Markup Language - HTML) y con las Hojas de estilo en cascada (Cascading Style Sheets - CSS).

Este libro es adecuado para cualquiera que tenga interés en aprender a desarrollar sus propias páginas web. No son necesarios conocimientos previos para aprender con este libro. Lo único que es necesario es saber utilizar un ordenador y saber navegar por la Web.

Contenido del libro

El contenido de este libro se estructura en tres apartados bien diferenciados:

Parte 1

En la primera parte del libro se trabajan conceptos generales que definen como funcionan en la web. Se explica conceptos de estructura física y estructural lógica de un sitio web. Se detalla cómo utilizar la estructura física en HTML y direcciones que se emplean a la hora de crear enlaces a otros sitios web. Parte 2

En la segunda parte se trabaja HTML. Partiendo de la estructura básica de una página web, se explican las etiquetas de HTML que se utilizan para definir el texto o los enlaces, las tablas, los formularios y otros elementos.

Parte 3

En la tercera y última parte se explica CSS el lenguaje que se emplean para definir el formato y la presentación de una página web. Se explica cómo utilizar el color, los tipos de letras, alineaciones, márgenes, bordes, fondos, transformaciones y transiciones con el fin de diseñar una página web.
